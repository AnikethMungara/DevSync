# DevSync

A collaborative development environment with real-time editing, code execution, and file management.

## Tech Stack

**Backend**
- Python 3.10+
- FastAPI - Asynchronous web framework
- Uvicorn - ASGI server
- aiosqlite - Async SQLite database adapter
- aiofiles - Async file I/O
- Pydantic - Data validation and settings management

**Frontend**
- Next.js 16 - React framework with Turbopack
- React 19 - UI library
- TypeScript - Type-safe JavaScript
- Zustand - State management
- Monaco Editor - VS Code editor component
- Tailwind CSS v4 - Utility-first CSS framework
- Lucide React - Icon library
- Sonner - Toast notifications

**Development Tools**
- Git - Version control
- Python subprocess - Code execution engine
- WebSocket - Real-time communication

## Quick Start

```bash
START.bat
```

This starts both backend (port 8787) and frontend (port 3000).

To stop: Type "end" or close the CMD windows.
